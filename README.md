
# Rust Obsidian Theme

A modern and clean Obsidian theme inspired by Rust programming language documentation aesthetics.

## Features

- **Rust-inspired color palette** - Warm oranges and earthy tones
- **Clean typography** - Optimized for long reading sessions
- **Dark mode support** - Easy on the eyes
- **Syntax highlighting** - Enhanced code block visibility
- **Responsive design** - Works seamlessly across all devices

## Installation

1. Open Obsidian Settings
2. Navigate to **Appearance â†’ Themes**
3. Search for "Rust Obsidian Theme"
4. Click **Install** and then **Use**

Or manually:

- Download the theme folder
- Place it in `.obsidian/themes/`
- Restart Obsidian and select from Appearance settings

## Customization

Edit the CSS variables in the theme files to customize colors and spacing to your preference.
